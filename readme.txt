README

Português

Este programa é um Gerenciador de Estoque desenvolvido em Python com interface gráfica Tkinter.
Ele foi pensado para facilitar o controle de materiais, produtos ou itens armazenados,
permitindo registrar entradas, registrar retiradas, consultar o estoque, filtrar itens por validade,
visualizar alertas de produtos próximos do vencimento ou já vencidos, exportar relatórios em PDF
 e manter um histórico de movimentações para acompanhamento e organização.

Principais funcionalidades:
- Cadastro de entradas de materiais com quantidade, validade e motivo.
- Registro de retiradas com histórico de movimentação.
- Consulta do estoque em uma visualização organizada.
- Filtros por nome e por situação de validade.
- Alertas automáticos de itens com validade próxima ou vencida.
- Exportação de relatórios em PDF.
- Troca entre temas claro e escuro.

Como executar:
Há duas formas simples de abrir o programa:

1. Pelo terminal:
   - Abra um terminal na pasta do projeto.
   - Verifique se o Python está instalado.
   - Execute um dos comandos abaixo:
     python app.py

   Se o comando python não funcionar no seu sistema, tente:
     py app.py

2. Pelo arquivo de inicialização:
   - Na pasta do projeto, execute o arquivo iniciar_estoque.bat.
   - Isso abrirá o programa automaticamente usando o Python instalado no sistema.
   - Em muitos computadores Windows, basta dar um duplo clique no arquivo para iniciá-lo.

Como usar o programa:
- Na aba "01 - Entrada", registre novos itens com nome, descrição, quantidade, validade e motivo.
- Na aba "02 - Retirada", selecione o material e registre a quantidade que saiu do estoque.
- Na aba "03 - Estoque", consulte o inventário, filtre por material ou validade e utilize os botões
  de entrada rápida, retirada rápida e retirada máxima.
- Na aba "04 - Log", acompanhe as movimentações do material selecionado.
- O programa salva automaticamente os dados do estoque, da lixeira e das configurações da interface.

English

This program is a Stock Management application developed in Python with a Tkinter graphical interface.
It was designed to make it easier to control materials, products, or stored items by allowing you to
record stock entries, register withdrawals, consult inventory, filter items by expiration date, view
alerts for products that are close to expiration or already expired, export reports to PDF, and keep
an activity history for better organization and tracking.

Main features:
- Registration of stock entries with quantity, expiration date, and reason.
- Registration of withdrawals with movement history.
- Organized inventory consultation.
- Filters by item name and expiration status.
- Automatic alerts for items near expiration or expired.
- PDF report export.
- Light and dark theme switching.

How to run:
There are two simple ways to open the program:

1. Using the terminal:
   - Open a terminal in the project folder.
   - Make sure Python is installed.
   - Run one of the commands below:
     python app.py

   If the python command does not work on your system, try:
     py app.py

2. Using the startup file:
   - In the project folder, run the file iniciar_estoque.bat.
   - This will start the program automatically using the Python installed on the system.
   - On many Windows computers, you can simply double-click the file to launch it.

How to use the program:
- In the "01 - Entry" tab, register new items with name, description, quantity, expiration date, and reason.
- In the "02 - Withdrawal" tab, select the material and record the quantity leaving the stock.
- In the "03 - Inventory" tab, view the inventory, filter by material or expiration status, and use
  the quick entry, quick withdrawal, and maximum withdrawal buttons.
- In the "04 - Log" tab, review the movements for the selected material.
- The program automatically saves the stock data, recycle bin data, and interface configuration.
