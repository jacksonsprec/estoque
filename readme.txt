README

Português

Este programa é um Gerenciador de Estoque desenvolvido em Python com interface gráfica Tkinter.
Ele permite registrar entradas e retiradas de materiais, consultar o estoque em tempo real,
filtrar itens por validade, visualizar alertas de produtos próximos do vencimento ou vencidos,
exportar relatórios em PDF e manter um histórico de movimentações.

Como executar:
1. Abra um terminal na pasta do projeto.
2. Verifique se o Python está instalado.
3. Execute o comando:
   python app.py

Se o comando python não funcionar no seu sistema, tente:
   py app.py

English

This program is a Stock Manager developed in Python with a Tkinter graphical interface.
It allows you to register item entries and withdrawals, consult inventory in real time,
filter items by expiration date, view alerts for items nearing expiration or already expired,
export reports in PDF format, and keep a history of movements.

How to run:
1. Open a terminal in the project folder.
2. Make sure Python is installed.
3. Run the command:
   python app.py

If the python command does not work on your system, try:
   py app.py
